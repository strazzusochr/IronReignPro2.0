🎯 METHODE 1: GITHUB ACTIONS (EMPFOHLEN!) ⭐
═══════════════════════════════════════════════════════════════════

✅ 100% AUTOMATISCH - Keine Installation nötig!
✅ KOSTENLOS - Immer!
✅ FUNKTIONIERT IMMER - GitHub baut für dich

📖 Anleitung: GITHUB_ACTIONS_EINFACH.md

⚡ Schnellstart:
   1. GitHub-Account erstellen
   2. Repository erstellen
   3. Dateien hochladen mit: github-upload.ps1
   4. 5 Minuten warten
   5. APK downloaden von GitHub Actions
   6. FERTIG!

⏱️  Einrichtung: 10 Minuten
⏱️  Build-Zeit: 3-5 Minuten
💰 Kosten: KOSTENLOS
🔧 Installation: NICHTS!


═══════════════════════════════════════════════════════════════════
🐳 METHODE 2: DOCKER BUILD
═══════════════════════════════════════════════════════════════════

✅ Vollautomatisch in isolierter Umgebung
✅ Reproduzierbar
✅ Für Entwickler

📖 Anleitung: AUTOMATISCHE_BUILDS.md (Docker-Sektion)

⚡ Schnellstart:
   1. Docker installieren
   2. docker build -t ironreign-builder .
   3. docker run -v "$(pwd)/output:/output" ironreign-builder
   4. APK in output/ironreign-debug.apk

⏱️  Einrichtung: 15 Minuten
⏱️  Build-Zeit: 2-3 Minuten
💰 Kosten: KOSTENLOS
🔧 Installation: Docker Desktop

💻 METHODE 3: LOKALER BUILD (WINDOWS)
═══════════════════════════════════════════════════════════════════

✅ Volle Kontrolle
✅ Offline möglich (nach erstem Build)
✅ Für Entwickler

📖 Anleitung: WINDOWS_BUILD_ANLEITUNG.md

⚡ Schnellstart:
   1. JDK 17 + Android Studio installieren
   2. PowerShell öffnen: .\build_windows.ps1
   3. Oder CMD: build_windows.bat
   4. APK in app\build\outputs\apk\debug\

⏱️  Einrichtung: 30 Minuten
⏱️  Build-Zeit: 1-3 Minuten (nach Setup)
💰 Kosten: KOSTENLOS
🔧 Installation: JDK 17 + Android Studio
